WHAT THE URL LOOKS LIKE:
=======================

Format:
  postgresql://user:password@hostname:5432/database

Real example:
  postgresql://admin:K8x9nJ2mL4p@db.railway.internal:5432/railway

Parts:
  - admin = username
  - K8x9nJ2mL4p = password
  - db.railway.internal = host
  - 5432 = port
  - railway = database name
