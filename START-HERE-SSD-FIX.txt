==============================================================================
                    SSD HIGH USAGE FIX - START HERE
==============================================================================

Your SSD was spiking to 99% due to:
  1. Windows Defender scanning dev folders constantly
  2. Windows Search using 2TB virtual memory
  3. Superfetch causing background disk activity
  4. Pagefile too large (24GB)
  5. C: drive only 28% free

==============================================================================
                   HOW TO FIX (2 MINUTES)
==============================================================================

STEP 1: Open PowerShell as Administrator
   Right-click → Run as Administrator

STEP 2: Navigate to your project
   cd E:\LRBillingOnline

STEP 3: Run the fix script
   .\FIX-ALL-SSD-ISSUES.ps1

STEP 4: Follow the prompts
   - Script will apply 8 fixes automatically
   - You'll need to restart your computer
   - You'll need to reduce pagefile manually after restart

STEP 5: Restart and verify
   - Open Task Manager → Performance → Disk
   - Should see 10-20% instead of 99%

==============================================================================
                        WHAT IT FIXES
==============================================================================

✓ Windows Defender - Adds exclusions for dev folders
✓ Superfetch - Disables background preloading
✓ Windows Search - Resets corrupted index
✓ Background Apps - Disables unnecessary processes
✓ SSD Optimization - Runs TRIM on all drives
✓ Temp Files - Cleans old temporary files
✓ Recycle Bin - Empties all bins
✓ Pagefile Warning - Alerts if too large
